# 執筆ガイドライン

本ガイドラインは `lore/` 配下の共通資料を踏まえ、本シリーズ固有の設定・ルール・プロット指針をまとめたものである。

---

## 1. シリーズ概要

**「嵐の城砦にて」**（仮題）

破砕戦争後、褪せ人到来前——最も希望のない時代のストームヴィル城。
末端貴族エラン・グレイフォードに上書き転生した現代人が、
接ぎ木の狂気に堕ちていくゴドリックの治世を内側から生き延びようとする政治劇・群像劇。

### 既存シリーズ（レイ編）との差別化
| | レイ編 | エラン編（本作） |
|---|---|---|
| 時代 | 黄金時代（繁栄期） | 破砕戦争後（暗黒期） |
| 舞台 | 学院・王都（権威の中心） | ストームヴィル城（辺境の荒廃した城） |
| 主人公の立場 | 師の庇護下にある弟子 | 狂気の主君に仕える末端貴族 |
| トーン | 日常系＋軽快 | 政治劇＋群像劇、重め |
| 主人公の前世知識 | 未来を知る苦悩 | 主君の末路を知る苦悩 |

## 2. 時代設定 — 破砕戦争後、褪せ人到来前

### 世界の状態
- エルデンリングは砕かれ、黄金律は機能不全に陥っている
- デミゴッドたちは大ルーンを得て各地に割拠している
- ゴッドフレイは追放され、褪せ人となっている（が、まだ狭間の地には戻っていない）
- 祝福の導きはまだ褪せ人たちに与えられていない
- 世界は荒廃し、各地で亜人や魔獣が跋扈している
- 死のルーンは盗まれ、「死に生きる者」が発生している

### ストームヴィル城の状態
- ゴドリックが領主として城を支配。黄金の一族の末裔を自称するが、他のデミゴッドからは軽んじられている
- 接ぎ木の実験が徐々に進行中。まだ大規模ではないが、城内には不穏な気配がある
- 城の兵士や家臣の士気は低い。ゴドリックへの忠誠は惰性と恐怖のあいだ
- リムグレイブ自体は肥沃な土地だが、城の統治は荒んでいる

### 存在するもの（黄金時代にはなかったもの）
- 褪せ人という概念（ただしまだ狭間の地に戻っていない）
- 陰謀の夜の結果（ゴッドウィンは死んでいる、ラニは肉体を捨てている）
- 「死に生きる者」の発生
- 破砕戦争の傷跡
- 各地のデミゴッドの割拠

## 3. 主人公：エラン・グレイフォード

### 基本情報
- 20代半ばの男性。リムグレイブの末端貴族グレイフォード家の当主（あるいは嫡男）
- 現代日本の人間がエランの身体に「上書き転生」。元のエランの記憶・人間関係・社会的立場をすべて引き継ぐ
- ゲーム「エルデンリング」をプレイ済み。ゴドリックの接ぎ木の狂気の末路、褪せ人による討伐を知っている
- 転生前のエランはゴマすり・処世術に長けており、ゴドリックからそれなりの信任を得ている。主人公はこの「遺産」を活用しつつ、内心では綱渡り

### 性格
- 内面：フランクで現代的。ツッコミ気質。だが根は善良で、見捨てられない性分
- 外面：貴族らしい慎重さと礼節。必要に応じてゴマも擦る（元のエランの処世術を記憶で引き継いでいる）
- ゴドリックへの感情：ゲーム知識では「哀れなボス」だったが、実際に仕えるうちに個人的な情が移る。「どうしようもないデミゴッドだが、死んでほしくない」

### 口調
- **一人称**：「私」（公的・フォーマル）/ 「俺」（内心モノローグ、ごく親しい相手）
- **二人称**：「閣下」（ゴドリック）/「卿」「殿」（貴族同士）/ 名前呼び・「あなた」（友人・親しい相手）
- **内心**：現代日本語のフランクな口調。「マジかよ」「やべえ」等も内心では出る
- **対外**：貴族相応の硬めの言葉遣い。「〜でございます」「〜かと存じます」等
- **親しい相手**：貴族にしてはやや砕けた程度。「〜だろう」「〜ではないか」
- **語尾の禁止**：内心以外で現代口語を使わない

## 4. 主要キャラクターと口調

### 接ぎ木のゴドリック
- `lore/キャラクター定義_黄金時代版.txt` の定義に従う
- 一人称「我（われ）」、二人称「お主」/「〜風情」
- 尊大な古語調。「我こそは〜」「〜なるぞ」
- **本作での描写方針**：単なる狂った暴君ではない。弱さゆえの虚勢、一族への劣等感、それでも領主であろうとする矜持を描く。接ぎ木への傾倒は徐々に進行させ、読者（とエラン）が「まだ引き返せるのでは」と思える段階を長く保つ。ただし「いい人」に寄せすぎないこと。公の場では端的かつ尊大、私的な場でのみ弱さを見せる。兵士への情は「黄金樹に連なる同胞」への義務感が根底にあり、純粋な善意ではない。故郷はローデイルであり、ストームヴィル城は仮の居城にすぎないという意識がある

### ケネス・ハイト
- リムグレイブの正統な血筋を自認する貴族。ゴドリックの統治に公然と批判的
- 一人称「私（わたし）」、二人称「貴公」「君」
- 気位の高い貴族口調。「〜であるぞ」「〜と言えよう」「恐れながら」
- 理想主義者だが実行力に欠ける。エランにとっては面倒だが無視できない存在
- 原作セリフ参考：「あの簒奪者めが、正統の血を汚しているのだ」

### エドガー
- モーンの城砦の城主。忠義と義務に殉じる武人
- 一人称「私（わたし）」、二人称「貴殿」「あなた」
- 落ち着いた武人口調。誠実で簡潔。「〜だ」「〜であろう」
- エランにとっては信頼できる同僚的存在。この時代の「まともさ」の象徴
- 原作セリフ参考：「私には、義務がありますので」

### ネフェリ・ルー
- ゴッドフレイの血を引く女戦士。一時的に城に逗留する程度の出番
- 一人称「私（わたし）」、二人称「あなた」
- 芯の強い女性口調。武人としての矜持

## 5. 文体とトーン

- **視点**：三人称（エラン視点）
- **地の文**：硬質で重厚。荒廃した世界の冷たさと、その中で生きる人間の温度を対比させる
- **トーン**：政治劇・群像劇。基調は重いが、エランの内心のフランクさが適度な息抜きになる
- **ジャンル**：ダークファンタジー × 宮廷政治劇

### 二重の語り
本作の文体上の最大の特徴は、エランの「外面」と「内心」の二重性。
- 地の文：硬質で叙情的。破砕戦争後の荒廃した空気を反映
- エランの内心：（）や独白で現代的なフランクさを挟む。ただし多用しすぎない
- この落差が本作のユーモアと緊張感を生む

### 避けるべきこと
- ゲーム用語（「ステージ」「ボス」「NPC」「HP」等）を内心でも使わない
- 前世知識を「便利なチート」として描くこと → 「主君の末路を知っているのに変えられるか分からない苦悩」
- ゴドリックを一面的な悪役にすること → 同情と嫌悪の間で揺れる複雑さを保つ
- 荒廃一辺倒 → リムグレイブの豊かさ、城内の人間模様など、暗さの中の光を描く

### 部の境界ルール（自動執筆エージェント向け）

1. **各話は必ず該当する部のエピソード表に従うこと。** エピソード表の「内容」を軸に展開し、勝手に飛ばしたり順序を変えたりしない
2. **部の最終話（ep008, ep018, ep030, ep040）では、終了条件を必ず満たすこと**
3. **次の部のキャラクター・イベントを先取りしないこと**（例：第一部でケネスとの対面を書かない）
4. **エピソード表のタイトルは「案」であり、内容に合わせて変更してよい**
5. **エピソード表の「内容」は必須要素。それ以外の描写は自由に補ってよい**

## 8. 世界観の深い設定

構造的な理解が必要な場合は `lore/世界設定ノート.txt` を参照する。特に：

- **破砕戦争の結果**：各デミゴッドの勢力圏、ストームヴィル城の孤立

## 9. 不確かな設定への対応

- 不確かな設定は使わない
- 原作ロアの解釈に幅がある場合、断定せず曖昧に留める
- 破砕戦争後の詳細な時系列は原作でも不明確な部分が多い。「長い年月が経った」程度の曖昧さを保つ
